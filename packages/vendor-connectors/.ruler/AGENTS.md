# vendor-connectors

Cloud vendor SDK wrappers with consistent interfaces.

## Dependencies

- `extended-data-types` (workspace)
- `lifecyclelogging` (workspace)

## Connectors

- **AWS**: S3, SSO, CodeDeploy, Organizations
- **Google**: Cloud, Workspace, Billing
- **GitHub**: Repos, Actions, Teams
- **Vault**: Secrets, Auth
- **Slack**: Webhooks, API
