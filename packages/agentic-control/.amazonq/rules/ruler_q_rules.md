# agentic-control

AI agent fleet orchestration toolkit (TypeScript).

## CLI Commands

```bash
agentic fleet list [--running]
agentic fleet spawn <repo> "<task>"
agentic fleet followup <id> "<message>"
agentic triage analyze <id> -o report.md
agentic tokens status
```

## Key Types

- `Fleet` - Agent fleet management
- `Triage` - AI-powered session analysis
- `TokenManager` - GitHub token switching
